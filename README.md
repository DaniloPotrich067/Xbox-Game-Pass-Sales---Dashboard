# Dashboard de Vendas — Xbox Game Pass

## Descrição do Projeto
Este projeto consiste na construção de um dashboard interativo em Microsoft Excel
para análise de vendas de assinaturas relacionadas ao ecossistema Xbox,
incluindo Xbox Game Pass, EA Play e Minecraft Season Pass.

O dashboard foi desenvolvido com foco em clareza visual, organização dos dados
e facilidade de interpretação, permitindo uma análise rápida dos principais
indicadores de vendas.

---

## Objetivo
Apresentar de forma visual e objetiva:
- O valor total arrecadado por tipo de assinatura
- A distribuição das vendas por produto
- A segmentação por tipo de plano (Monthly, Quarterly, Annual)
- O status das assinaturas (ativas ou não)

---

## Dados Utilizados
Os dados utilizados neste projeto são fictícios e simulam um cenário real de vendas,
contendo as seguintes informações:
- Produto (Xbox Game Pass, EA Play, Minecraft Season Pass)
- Tipo de assinatura (Mensal, Trimestral, Anual)
- Status da assinatura (Yes / No)
- Valores em reais (R$)

Esses dados foram organizados e tratados diretamente no Excel.

## Como Reproduzir o Projeto
1. Faça o download do arquivo `Dashboard.xlsx`
2. Abra o arquivo no Microsoft Excel (2019 ou superior) ou no Excel Online
3. Acesse a aba **Dashboard**
4. Utilize os filtros disponíveis para interagir com os dados

---

## Observações Importantes
- O modo de tela cheia completo depende de uma licença ativa do Microsoft 365
- Em contas gratuitas, recomenda-se utilizar o modo tela cheia do navegador (F11)
- O projeto não utiliza macros ou código VBA
